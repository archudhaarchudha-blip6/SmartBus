# 🚌 SmartBus

## Edge AI-Based Smart Rural Bus Monitoring and Passenger Density Prediction System

### 📌 Project Overview
SmartBus is an AI-powered smart public transportation system designed to improve rural bus services. It helps passengers track buses, report delays, predict passenger density using Edge AI, and provides intelligent recommendations to transport authorities.

### ✨ Features
- Live Bus Tracking
- Bus Delay Reporting
- GPS Monitoring
- Passenger Density Prediction
- AI Recommendations
- Admin Dashboard
- Emergency Alerts
